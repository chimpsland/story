遇到流浪画家住到店里，这种情况怎么办？

一个朋友求助，他在一个村边路旁开了一间旧书小店，在一个算是历史悠久的风景区的地方，才开店没有两周，这里不是人烟稀少的地方。

今天下午一个女人来聊天，说自己是画家在流浪。女人貌似衣着得体，谈吐也像是懂油画。

下面我模糊掉一点身份信息， 称我朋友为 X， 女画家为 P （Painter).

当 X 听说 P 画家在流浪的时候就同意她在店面地上睡一下。

我朋友就帮她准备了两张床板，就是两块大木板。 说明一下，他是完全没有什么两性方面的企图，只是一时好心同意了， 其实 X 的小店完全不适合收留流浪者。 然后才知道她什么都没有了，手机钱包什么的都没有了。 然后她的记忆也像是很有问题，过了两个小时又开始咳嗽。 X 请她吃了晚餐， 他自己也是很简陋的晚餐， 同时给她买了一只新碗一只新勺子，因为 X 小店里多余的碗也没有。

而这时村里一个捡垃圾的村民就很有兴趣的专门进来看这个女画家，看了好久并且保持着一种奇怪的神秘微笑。 这时候 X 就心里有点奇怪，捡垃圾的本地村民似乎认识 P， 或者发现了什么似的。 但是垃圾汉脑子也有点问题，也不说什么，也不说他是不是认识事宜，垃圾汉在店门口站了好久才走。

这种情况怎么办？

半天的时间她都在说曾经死过几次，一问又不说了，不问的时候又开始说。 夹杂着大量的中医的神奇力量，各种穴道， 有骨骼上刺激骨髓的穴道，还有可以治愈癌症的火罐 。 X 不懂多少中医，因此听得有点心不在焉。 但是拼凑其中零碎的信息可以知道另一件事：

她的父母都 是东北下岗工人，原来是大型军工企业，后来工厂不断转产。

所以她的故事背景是一个史学研究只会轻轻带一笔的事件， 在上世纪90年代中国改革开放的同时， 东北原有的重工业体系实际上在破产。 一间工厂的衰落只是这个消失的次旋律中的小事， 而一个家庭的破落就是这件小事中的微尘。

当她的父母一代的生活越来越拮据的时候， 她身为下岗工人的下一代其实是以两代人的命运走过了历史潮流中的一个小小的漩涡。

然而，也许在她的家庭中没有人可以建立正确的宏观视野， 更不要说看清楚左右自己命运的主要力量。 虽然也不能将她的悲剧全部归功于上一代东北工人的下岗大潮， 但是一片工厂的没落并不仅仅是一片厂房的荒废， 情况更像是一条条沉没的客轮， 小工厂带沉几百个家庭，大工厂带沉过万个家庭。

而当时东北的下岗工人何止千万。

在过去一个世纪东北工业经历日本控制的满洲国时期，成为亚洲工业重地。 1949年新中国后，在瓦房店式的衰落中迭加一次次的政治运动，生产成本早就堆高， 竞争力落后，如果在世界上倒着数恐怕是比较领先的。 如果生产成本过高当然就是一种浪费， 那么，不生产就不会浪费。 问题是：工人和他们的家庭难道都是机器，一拉闸就断电停机？ 三十年里工人阶级在宣传中是国家的领导者， 在中国经济转型时却轻松的就被下岗了。

P 本人的性情当然可能是悲剧的内因。 我们不可以将所有人的命运归结到历史背景中， 但是恰好，她和她的家人就是在这个东北工业衰落的潮流中。

当中国南方轻工业加速发展时，东北实际上就是代价的一部分。 2000年的时候中国加入 WTO ，以出口加工业为主， 南方轻工业生产的日用品为主力推动中国制造业成为世界工厂时， 当时成千上万东北下岗工人和他们的子女就已经置身在另一个历史漩涡中了。

所以在十年前有一种可以算是半官方的说法是中国经济必需保持 8% 的年增长率， 原因就可以由这一件小事体现出来，中国经济发展的代价太大了， 当时认为 8% 才能抹平这些代价。 现在显然已经有识趣的学者重新说明新的发展和其它原理。

显然 P 是不知道她的命运不仅仅是从小时候生活的工厂开始的， 而是东北很多人分享的一个愚蠢政府和政黨的无良大甩锅行为甩出来的一条细细的面条。

这个问题下，各位朋友给的建议， 不论有没有转告成为 X 的可选方法， 都是不同角度的思路拓展，谢谢。 现在情况似乎还是可以控制的。

下午，女画家推着一部共享单车走了， 她说如果路过还会来。这部共享单车是她重要的工具， X 想给她一个车锁，她谢绝了，共享单车上的车锁会引人怀疑。 之前 X 想给她一条睡袋，她也谢绝了，画家害怕自己钻入睡袋后会被人从外面捆住。

另一位村里的朋友想要提供给她一套被褥，她也谢绝了。

对于小店主 X，他是一个值得批斗的人物。 虽然现在他也需要心理辅导的样子， 他本人因为这么一件小事也遭遇了心理重击。 但是以后他值得好好教育一下：他还以为自己可以保留什么善意， 他不知道自己根本没有能力、没有资格、不配做什么好人。

一个小破店，没有一点承担能力，还收留流浪女？ 结果是，根本就是会对女画家造成又一次心灵伤害！

他比乞丐们都不如， 更不要说乞丐中的老头， 虽然这种事实不会让他自己挂怀， 可是在流浪画家眼里恐怕还就是这样。

这个社会中，还是有不少人傻到不知道自己才是傻子，包括我自己吧。 看了几万年的历史却看不透村里的垃圾汉用鼻子一闻就能闻出来的事情。

最后希望女画家在困难的时候能回来休息一下，这里至少有 Clean Water and Shelter.
